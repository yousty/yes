Overview

Creates and registers event classes for aggregate attributes

This class resolver generates event classes that represent state changes in aggregate attributes. Each event class is automatically configured with a schema that validates the required attributes and their types.

Examples:

Generated event class structure

class UserEmailUpdated < Yes::Core::Event
  def schema
    Dry::Schema.Params do
      required(:user_id).value(Yousty::Eventsourcing::Types::UUID)
      required(:email).value(String)
    end
  end
end
