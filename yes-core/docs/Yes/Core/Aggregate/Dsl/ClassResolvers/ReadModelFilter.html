Overview

Creates and registers read model filter classes for aggregates

This class resolver generates filter classes that provide query scopes for read models. Each filter class is automatically configured with basic scopes and a reference to its corresponding read model class.

Examples:

Generated read model filter class structure

class UserFilter < Yousty::Eventsourcing::ReadModelFilter
  has_scope :ids do |_, scope, value|
    scope.by_ids(value.split(','))
  end

  private

  def read_model_class
    Test::User::Aggregate.read_model_class
  end
end
